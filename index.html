        About This Project

        This project is my high school independent study for Physics. To complete my goal of graduating an entire year early, with Honors and LISA diplomas, I needed to do extra classes outside of school. I have done a couple other independent studies in the form of websites (see my github profile), but this one is my best so far. (I've had a lot more experience now)

        I have taken the entire class and split it up as is on the syllabus, so it is split into Units, which are then split into Topics. I will cover everything you need to know about Physics in each Topic, but I will not be showing any shortcuts. I may or may not have coded the website with said shortcuts, but I emplore you to figure them out on your own. I do this because I don't want to be blamed for teaching anything incorrectly, and because I don't know what works best for you; only you can figure that out. So please, play around with the formulas and whatnot down there, don't just read this as any boring old reference book.

        Metric to Metric Conversions

        The Metric System is a measurement system in base 10. This means that every unit of a certain measurement is some factor of 10 in ratio to another of the same measurement. Shown below is a quick table of common metric conversions (replace "meter" with whatever unit you like), which both serves as the definition for metric units and the conversion to the base unit (in this case, meters). When converting between any units, the generally accepted way is to use a fraction defined by the ratio between the unit you'd like to get and the unit you have. The calculator further below shows this very nicely. I will not show any shortcuts, as I have mentioned before, but I will mention that one exists using one of the laws of exponents. (You can find it on my Trigonometry Website [It really sucks compared to this site, doesn't it?]).
